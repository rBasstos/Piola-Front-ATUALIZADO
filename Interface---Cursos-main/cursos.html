Curso 1 - Java

O curso abrange desde os conceitos iniciais até tópicos avançados em Java, incluindo a introdução à linguagem e sua instalação. Serão explorados conteúdos como tipos primitivos, manipulação de dados, operadores e estruturas, bem como o desenvolvimento de interfaces gráficas com Swing e JavaFX, manipulação de vetores e criação de métodos, além de exercícios diversos.

Curso 2 - JavaScript

Os alunos serão capacitados a dominar os conceitos essenciais da linguagem JavaScript, desde sua sintaxe básica até manipulação de arrays, estruturas condicionais, laços de repetição, temporizadores, classes, manipulação de datas e JSON. Com a inclusão de um projeto prático, os estudantes terão a oportunidade de aplicar suas habilidades de programação em cenários do mundo real.

Curso 3 - HTML e CSS

Os alunos serão guiados por uma série de aulas abrangentes que cobrem desde a introdução aos fundamentos do HTML e CSS até a criação de um projeto web completo. O curso introduz o tema e prepara o ambiente de desenvolvimento. Em seguida, aprenderemos a cria e estilizar elementos como cabeçalho, search, menu de navegação, caixas de destinos, destaques e rodapé.

Curso 4 - C

O curso abrange tópicos essenciais para programadores novatos em C, iniciando com a instalação da IDE CodeBlocks e criação do primeiro programa. Em seguida, há uma introdução a tipos de dados e declaração de variáveis. Os alunos aprenderão sobre operadores, desvios condicionais, estruturas de repetição, funções, arrays, ordenação com Bubblesort, matrizes e ponteiros.
